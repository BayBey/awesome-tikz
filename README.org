- [[https://www.ctan.org/pkg/visualtikz][VisualTikZ]]       - Visual help for TikZ based on images with minimum text

** Gallery

- [[http://www.texample.net/tikz/][TeXample]]                    - Gallery of wonderful examples made by TikZ

  Thoma
